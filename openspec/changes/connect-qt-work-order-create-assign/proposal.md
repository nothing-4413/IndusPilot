# Change: Qt 客户端维护工单新建与分派

## Why

Qt 客户端已经可以查看维护工单并执行开始、完成、关闭流转，但仍无法在桌面端创建新工单或分派责任人。工业运维支持系统需要从告警/诊断建议进入实际处置，因此客户端应补齐工单创建和分派入口。

## What Changes

- 维护工单页新增创建工单入口，提交 `id`、`assetId`、`summary`，并支持可选 `alertId` 和 `assignee`。
- 维护工单页新增分派入口，选中工单后提交责任人。
- 工单列表增加摘要列，便于操作员确认工单内容。
- 创建或分派成功后刷新工单列表；后端不可用时保留离线演示数据和中文提示。

## Non-Goals

- 不在本次实现从告警表格一键生成工单。
- 不实现工单编辑、删除、批量分派或附件上传。
- 不改变后端工单状态机和权限模型。
