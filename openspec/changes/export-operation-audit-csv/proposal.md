# Change: 操作审计 CSV 导出

## Why

操作审计已经支持查询、筛选和分页，但管理复核、变更留痕和事故复盘通常需要可交付的导出文件。导出能力必须由独立权限保护，避免普通操作员批量带走审计数据。

## What Changes

- 新增 `audit:export` 权限，默认仅管理员角色具备。
- 后端新增 `GET /api/v1/audit/events/export`，复用审计筛选条件并返回 CSV 文件内容。
- 导出操作会写入 `operation-audit.export` 审计事件，记录导出人和导出数量摘要。
- Qt “操作审计”页面新增导出按钮，按当前筛选条件从后端下载 CSV 并保存到本地。
- HTTP 冒烟测试覆盖导出权限拒绝、管理员导出和 CSV 内容。

## Non-Goals

- 不实现异步大文件导出任务。
- 不实现导出文件加密、水印或审批流。
- 不改变审计记录保留周期和归档策略。
