# Change: 实现维护工单 HTTP API

## Why

告警生命周期已经可用，但维护工单仍只有列表入口。为了把告警处理闭环到维修执行，需要提供工单创建、告警转工单、分派、处理、完成、关闭和资产维修历史 API。

## What Changes

- 扩展工单服务，支持详情查询和按资产、告警、状态筛选。
- 新增工单创建、告警转工单和生命周期流转 HTTP API。
- 新增资产维修历史 HTTP API。
- 扩展集成测试和开发文档。

## Non-Goals

- 不实现排班、备件、工时和 SLA 计算。
- 不实现移动端离线工单同步。
- 不接入外部 ERP/MES 工单系统。
