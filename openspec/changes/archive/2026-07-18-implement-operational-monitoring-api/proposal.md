# Change: 实现运行监控 HTTP API

## Why

工业资产模块已经提供设备主数据入口，但运行监控仍停留在模块内存占位。为了支撑监控看板、告警触发、工单关联和 AI 诊断，需要把短生命周期运行状态作为独立能力接入 HTTP API，并与资产主数据保持分离。

## What Changes

- 扩展监控服务，支持运行状态列表、单资产查询、汇总和状态写入。
- 新增运行状态 HTTP API，覆盖查询、详情和写入场景。
- 增加 `monitoring:write` 权限，区分运行态写入和资产主数据维护。
- 扩展集成测试和开发文档。

## Non-Goals

- 不实现真实 PLC/OPC UA/MQTT 采集协议。
- 不实现 WebSocket/SSE 推送通道的真实长连接。
- 不把运行态数据持久化进 MySQL；本阶段仍保持与资产主数据分离的内存运行态。
