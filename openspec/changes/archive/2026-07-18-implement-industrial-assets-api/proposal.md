# Change: 实现工业资产 HTTP API

## Why

工业资产是运行监控、告警、维护工单和 AI 诊断的共同业务锚点。当前项目已经定义了资产领域模型、仓储边界和基础 HTTP 入口，但资产生命周期、层级查询和仓储化服务尚未形成完整闭环。

## What Changes

- 将资产服务改为依赖 `AssetRepository`，保留内存仓储作为本地开发默认实现。
- 补齐资产创建、详情、列表筛选和生命周期状态变更 HTTP API。
- 增加集成测试覆盖未授权、权限拒绝、创建、查询、筛选和状态变更。
- 更新后端开发文档，说明资产 API 的请求格式和权限要求。

## Non-Goals

- 不接入完整 SCADA/MES 实时采集协议。
- 不实现资产图片、附件、备件库存或点检计划。
- 不要求本阶段必须连接真实 MySQL 才能通过本地测试。
